Präsentation

Fügen Sie hier eine Einleitung/Präsentation mit den allgemeinen Konzepten zum Thema ein.

So erstellen Sie den die Kopfzeile der Seite:

  • Klicken Sie auf die Schaltfläche "Mise en page" (Layout) in der Symbolleiste.
  • Fügen Sie einen Bereich hinzu.
  • Verschieben Sie ihn an den Anfang der Seite.
  • Wählen Sie die 4. Layoutoption: "section à 2 colonnes avec encadré droit" (Bereich mit 2 Spalten und geradem Rahmen).
  • Fügen Sie in der linken Spalte das Makro "Droits concernés" (Betroffene Berechtigungen) ein.
  • Kopieren Sie den nebenstehenden Bereich mit dem Inhaltsverzeichnis und den Links unter „Verwandt“ in die rechte Spalte.

Alle Makros können leicht auf einer Seite kopiert und auf der nächsten eingefügt werden. Um die Makros zu kopieren, wechseln Sie in den Seitenbearbeitungsmodus.
Auf diese Weise können Sie die Makros dieser Seite übernehmen, z. B. den rechten Bereich mit dem Inhaltsverzeichnis und den Links, und müssen anschließend auf der Zielseite nur die Links bearbeiten, das Inhaltsverzeichnis ist bereits konfiguriert.

Für einen Titel: Geben Sie den Text ein und wählen Sie dann den Stil in der Symbolleiste oberhalb des Bearbeitungsbereichs oder verwenden Sie seine Tastenkombination: Strg+ die gewünschte Stufe, d.h. Strg-2 für Stufe 2, Strg-3 für Stufe 3 usw.

Die Stufen 2 und 3 sind nach Formatvorlage nummeriert.

Wenn Sie einen Zeilenumbruch einfügen, wird für den nächsten Absatz automatisch der Stil "Absatz" gewählt.

Der Stil "Absatz" gilt als Stufe 0, er entspricht der Tastenkombination Strg-0.
Um einen Zeilenumbruch einzufügen, ohne den Absatz zu wechseln, verwenden Sie Umschalt-Eingabetaste.

Um einen Hypertext-Link einzufügen, geben Sie das Zeichen [ ein: Die automatische Vervollständigung schlägt dann die Seiten des Confluence-Bereichs oder die Optionen "Rechercher" (Suchen) oder "Insérer un lien Web" (Web-Link einfügen) für einen externen Link vor.

Um ein Zitat einzufügen, wenden Sie den Stil "Anführungszeichen" über die Symbolleiste oder mit Strg-8 an.

Ein neuer Absatz wird im selben Stil verfasst. Um zum Standardstil zurückzukehren, wählen Sie erneut den Stil "Absatz".

Der vorformatierte Text entspricht der Tastenkombination Strg-7.
Wie bei Zeilen wird bei einem einfachen Zeilenumbruch im selben Stil weitergeschrieben, 
bei einem Absatzwechsel wird erneut der Standardstil gewählt.

Vorsicht bei zu langen Zeilen im vorformatierten Modus: Der Browser fügt keinen automatischen Zeilenumbruch ein, sondern versieht den Block mit einer horizontalen Bildlaufleiste.

Makros

Die meisten Elemente (Blöcke, Bilder, Inhaltsverzeichnisse usw.) können wie Makros auf zwei Arten eingefügt werden:

  • Verwenden Sie in der Symbolleiste das Menü "Insérer" (Einfügen); wenn das gewünschte Makro nicht in den Verknüpfungen zu finden ist, wählen Sie "Autres macros" (Weitere Makros) im Menü und suchen Sie es mit dem Makro-Browser.
  • Geben Sie direkt in das Eingabefeld das Zeichen "{" ein; die automatische Vervollständigung hilft dann, das gewünschte Makro zu finden.
    Die automatische Vervollständigung arbeitet nicht nur mit den ersten Zeichen, sondern durchsucht alle Makros. Sie funktioniert auch mit verschiedenen Sprachen und sucht im Originalnamen und in der aktuellen Version: Wenn die Confluence-Oberfläche zum Beispiel auf Französisch ist, finden Sie das Makro "Avertissement", indem Sie "Aver..." oder "Warn..." (für "Warnung") eingeben.

Hervorhebungsblöcke

Für jeden Block sind der Titel und das Symbol optional.
Das Bild ist nicht editierbar, aber der Titel wird vom Verfasser eingegeben.

Ein Block {warning}

Dies ist eine Warnung/Warning.

Ein Block {note}

Dies ist ein Hinweis/note um auf etwas hinzuweisen, ohne dass ein Problem vorliegt.

Ein Block {info}

Dies ist eine zusätzliche Information/Info.

Ein Block {tip}

Dies ist ein hilfreicher Tipp/tip.

com.atlassian.confluence.setup.DefaultBootstrapManager@2c204456

Dies ist ein interner Hinweis für andere Verfasser, der vor der Veröffentlichung der Seite gelöscht werden muss.

Bilder

Einfaches Bild

Fügen Sie das Bild über das Menü "Insérer" (Einfügen) oder mit der Tastenkombination Strg-Umschalt-M ein.

Bild mit Bildunterschrift

Das Makro "Bildunterschrift" wird beim Hinzufügen eines Bilds (und nur eines Bilds) im Inhaltsblock verwendet. Die Bildunterschrift wird unter dem beim Einfügen abgefragten Parameter Bildunterschrift eingegeben und kann anschließend durch Doppelklick auf das Makro oder durch einen einfachen Klick und "Modifier" (Ändern) abgerufen werden.

Die Bildunterschrift kann nicht leer sein. Um ein Bild ohne Bildunterschrift einzufügen, verwenden Sie das einfache Bild.
